# Communication Templates

All contributor communication goes through Discord. These templates ensure consistency.

## After Creating a PR

```text
@maintainers I have raised PR #[number].
Please review and let me know the expectations.
Link to PR: [URL]
```

**Post in:** `#development` channel

## After Creating an Issue

```text
@maintainers I have raised issue #[number].
Please review and let me know if this is valid.
Link to Issue: [URL]
```

**Post in:** `#development` channel

## Asking for Help

```text
@mentor I'm working on issue #[number] and need help with [description].

What I've tried:
1. [Step 1]
2. [Step 2]

Error/blocker: [description]
```

**Post in:** `#help` channel

> Contributors MUST explain what they've tried before asking for help.

## Requesting Issue Assignment

```text
@maintainers I'd like to work on issue #[number].
I plan to approach it by [brief approach description].
Can I be assigned?
```

**Post in:** `#development` channel

## Reporting a Blocker

```text
@maintainers I'm blocked on PR #[number].
Blocker: [description]
I've been waiting for review since [date].
```

**Post in:** `#development` channel
